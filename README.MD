# Mein Benutzer-Filter Programm

Hallo! Das ist mein kleines Python-Skript. Ich habe es geschrieben, um eine Liste von Benutzern (aus einer JSON-Datei) ganz einfach zu durchsuchen.

## Was kann das Programm?
Du kannst damit eine Datei namens `users.json` durchsuchen nach:
* Dem **Namen** eines Benutzers.
* Dem **Alter** eines Benutzers.
* Der **E-Mail-Adresse** eines Benutzers.

## Was brauchst du dafür?
1. Du musst **Python** auf deinem Computer installiert haben.
2. Du brauchst eine Datei namens `users.json` im gleichen Ordner wie das Skript.

## Wie benutzt man es?
1. Öffne dein Terminal oder deine Konsole.
2. Starte das Programm mit diesem Befehl:
   ```bash
   python filter_users.py
